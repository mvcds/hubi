Domain File(A.K.A Configuration File)

An YAML file containing information about a particular piece of the ubiquitous language.

Not all domain files translate to source files due to the "abstract" configuration e.g. this very one!

Ubiquitous Token(A.K.A Token, Domain Object, Object Value, Entity)

A piece of the ubiquitous language extracted from its domain file

It represents a phrase, expression, term, etc

  • name: How people refer to the token(Should be unique across the domain to avoid ambiguity)
  • description: A short sentence to teach/remind everyone what the token means
  • attributes: A collection of attributes a token is associated with.This collection is specially useful for developers and is where translators are expected to work more often.
  • filePath: The path to its file domain
  • abstract(false): A flag to inform translators if they should or not translate the token
  • comment: An extra message, for developers, about the token
  • aliases: Alternative names, abbreviations, etcOther ways to refer to the token itself
Attribute(A.K.A Characteristic, Member)

A token's characteristic.

Programming-wise, it is an object's member.

  • name: How the attribute is referenced relativelly to its token
  • type("string"): What kind of token it is
  • required(false): A flag to mark the attribute as required
  • description: A short sentence to teach/remind everyone what that attribute means
  • comment: An extra message, for developers, about the attribute
Translation(A.K.A Result)

Holds information relevant to create/analyse source files in given translation

  • lexicon: The translation's vocabulary which is a collection of token-like objectsIt is a map to be more precise
Translator(A.K.A Plugin)

Interpret the ubiquitous language's tokens in order to write source files automatically.

  • interpretToken: How each ubiquitous language's token is interpreted during translation("Interface")
Ubiquitous Language(A.K.A Domain Language)

A common language for everybody

Each translator will interpret its token diffetently

  • tokens: A collection of tokens
