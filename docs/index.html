Domain File⚠It does not map to a source file

An YAML file containing information about a particular piece of the ubiquitous language which will be transformed into a token

Ubiquitous Token

A piece of the ubiquitous language (phrase, expression, term, etc) extracted from the domain file

  • name: How people refer to the token (should be unique across the domain to avoid ambiguity)
  • description: A short sentence to teach or remind everyone what that token means
  • attributes: A collection of attributes a token is associated with (this collection is specially useful for developers)
  • filePath: The path to its file domain
Attribute

A token's characteristic

  • name: How the attribute is referenced relativelly to its token
  • type("string"): What kind of token it is
  • required(false): A flag to mark the attribute as required
  • description: A short sentence to teach or remind everyone what that attribute means
Translator

An object which translates the ubiquitous language's tokens relevant source files for developers

  • ubiquitousLanguage: The language the translator will work on
  • tokens: The tokens the translator will work on⚠Use "ubiquitousLanguage" instead
